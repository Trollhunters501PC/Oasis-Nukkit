# Documentacion de modifyWorld del JSON
la opcion "modifyWorld" del json se usa para modificar mundos!
ejemplo de uso:
```json
{
    ...
    "modifyWorld": [{ "worldName": "NombreDelMundo", "id": 1, "pos": [0, 0, 1]}...],
    ...
}
```
es un array de bloques para poner en el mundo.
worldName: nombre del mundo donde se pondra el bloque
id: id del bloque en numero
pos: posicion del bloque [x, y, z]
